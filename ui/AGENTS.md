# ui/AGENTS.md

## Mission
Research and implement readable, modern, musician-friendly UI.

## Scope
- Color theme research and accessibility-minded palette selection.
- Typography scale and spacing tuned for legibility.
- Layout hierarchy and visual grouping for fast scanning while performing.
- Theme extension patterns for future branded styles.

## Standards
- Prioritize contrast and readability over novelty.
- Keep labels, controls, and values visually consistent across tabs.
- Validate desktop and resizable-window behavior.
