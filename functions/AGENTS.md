# functions/AGENTS.md

## Mission
Own functional behavior of the app.

## Scope
- Drag/drop contracts and chain connectivity behavior.
- Button actions, state transitions, and event wiring.
- Runtime calculations and recommendation application logic.
- Control sizing behavior tied to functionality (click/drag affordance quality).
- Persistence and validation flows.

## Quality Bar
- Behavior must be deterministic and testable.
- Avoid UI regressions caused by logic changes.
- Keep function boundaries explicit and reusable.
